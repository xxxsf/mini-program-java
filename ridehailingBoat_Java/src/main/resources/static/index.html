Pinche API is running

Welcome! This backend exposes REST endpoints under /api/*. Try these sample calls:

Sample Endpoints

  • Info list (type 1): /api/info/lists?type=1&page=1
  • Dynamic list: /api/dynamic/getList?page=1
  • Notices (requires an existing id): /api/notice/index?id=1
  • Messages (requires sk and type): /api/msg/get?sk=mock_sk_xxx&type=1&page=1
  • My appointments (requires sk): /api/appointment/my?sk=mock_sk_xxx

Note: some endpoints expect existing data in the database and/or a valid sk token.

Status

  • Port: 8080
  • Database: MySQL (schema pinche)
